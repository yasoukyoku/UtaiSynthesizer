# UtaiSynthesizer

**Voice synthesis DAW — piano-roll singing synthesis (ScoreToCV → SVC), AI cover rendering,
source separation, and on-device training. Tauri 2 + Rust + ONNX Runtime, Windows.**

歌声合成 DAW:钢琴卷帘写谱直接合成人声、AI 翻唱渲染、人声分离、本地训练,全流程一个程序。
